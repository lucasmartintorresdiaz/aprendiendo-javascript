# Tema 1: JavaScript en HTML

En esta primera parte del curso aprendí y practiqué las tres formas principales de usar JavaScript en HTML:

1. **JS Inline**: Escribir JavaScript directamente en un atributo HTML, como `onclick`.  
2. **Etiqueta `<script>` interna**: Colocar el código JavaScript dentro del mismo archivo HTML usando `<script>`.  
3. **Archivo JS externo**: Separar la lógica JavaScript en un archivo independiente (`script.js`) y enlazarlo desde el HTML.  

Este tema muestra cómo integrar JavaScript con HTML de manera progresiva, desde lo más simple hasta lo más profesional.
